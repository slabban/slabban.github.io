State Estimation with YOLO

Project Overview

This project was a deep dive into the world of Multi-Object Tracking (MOT) where I employed the powerful Extended Kalman Filter (EKF) and an Intersection Over Union (IoU) association approach to track objects in a video stream. I deployed the algorithm in the Robot Operating System (ROS) middleware, which allowed me to tap into the vast ecosystem of ROS packages and tools.

Inspiration

My inspiration for this project came from this article by Jeremy Cohen. I had come across it over a year ago and thought: "I know how to design a Kalman Filter and have worked on Multiple Object Tracking, I should give this a shot." So, I did!

Results and Comparisons

I experimented with both YOLOv3 and YOLOv4, comparing results as seen in the video above. While YOLOv4 was better at association, YOLOv3 performed detections faster, which allowed the Kalman Filter to produce smoother estimates.

Using YOLOv3, I achieved a detection output rate of 17 HZ and an estimate rate of 50 HZ on my trusty 2017 laptop with a 1060 GPU. Meanwhile, YOLOv4 reduced the rate to 7.5 HZ but produced more accurate results. Iâ€™m curious how better-calibrated input images could improve performance further.

Multi-Object Tracking Algorithm Breakdown

My algorithm, like most MOTs, follows these five key steps:

  • Detection
  • Association
  • Track Identity Creation & Destruction
  • Estimation

Challenges and Alternatives

There are some drawbacks to classical approaches like the Extended Kalman Filter, such as tracking loss through ID swaps during occlusion. Approaches like SORT, which use the Hungarian Algorithm for association, can mitigate these issues. For even more robust solutions, DeepSort leverages feature maps from a Recurrent Neural Network (RNN) for association. You can learn more about DeepSort here.

Conclusion

This project was a fantastic learning experience. The open-source community has taught me a lot, and I hope this project can contribute in return and inspire others to take on similar challenges!
